# **App Name**: DerivPay

## Core Features:

- Account Linking: Securely link users' Deriv accounts to the app using API authentication to allow the app to manage deposits.
- Deposit Initiation: Allow users to specify the deposit amount and trigger a deposit request to their linked Deriv account.
- MPESA STK Push: Integrate MPESA's STK push functionality to send a payment request to the user's phone for PIN confirmation.
- Transaction Confirmation: Monitor the MPESA transaction status and automatically confirm the deposit to the user's Deriv account upon successful payment.
- Transaction History: Provide a transaction history for users to review their past deposit transactions.
- Customer Support Chatbot: AI chatbot tool answers users questions using Deriv documentation, deposit tutorials, and troubleshooting guides. Generates appropriate responses and guides for a variety of deposit problems.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for trust and financial transactions.
- Background color: Light cyan (#E0FFFF) to maintain a clean and airy feel.
- Accent color: Sea green (#2E8B57) for confirmations and success messages.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, readable UI.
- Use flat, minimalist icons to represent different transaction types and statuses.
- A clean and intuitive layout with a focus on usability for quick deposits.
- Subtle animations to indicate transaction processing and confirmations.
